1. 임의의 세 숫자 입력받기

      1. 숫자 외의 다른 문자를 입력 받으면 에러
      2. 3개의 숫자가 아니면 에러
      3. 서로 다른 숫자가 아니면 에러
   2. computer 서로 다른 임의의 세개 숫자 랜덤 뽑기
2. ball과 strike 갯수 구하기
   1. computer에 사용자가 입력한 숫자가 있는지 확인
      2. 있다면 위치까지 같은지 확인 (리스트의 indexOf)
   
3. replay

      2. play == 2라면
