## 기능 목록

### 게임 시작
- [X] 시작 메시지 출력

### 게임 진행
- [X] 입력 안내 메시지 출력

- [X] 정답이 될 랜덤 숫자 생성

    • 1부터 9까지의 숫자로 중복 없이 구성된 3자리 랜덤값 생성

- [X] 사용자로부터 숫자 입력 받기

    • 입력이 아래 조건에 해당하지 않을 시 예외 처리

    1. 숫자인가?

    2. 3자리 수인가?

    3. 중복 숫자가 없나?

    4. 1에서 9 사이에 있나?

- [X] 정답과 입력을 비교해 스코어 출력

    • 숫자만 일치 시 ball

    • 숫자와 자리 모두 일치 시 strike

- [X] 정답 시 게임 종료

### 게임 종료
- [X] 재시작 안내 메시지 출력

- [X] 재시작 여부 입력 받기

    • 1 입력 시 : 재시작

    • 2 입력 시 : 게임 종료

    • 이외의 값들은 예외 처리
