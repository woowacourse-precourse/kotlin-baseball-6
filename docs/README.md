# 구현할 기능 세부 목록

### 0. 시작
- 처음 시작시 `숫자 야구 게임을 시작합니다.` 문구 출력

### 1. 상대방(컴퓨터)의 수 생성
- `camp.nextstep.edu.missionutils`에서 제공하는 `Randoms`을 사용하여 3자리의 랜덤 수 생성
- 생성된 수가 서로 다른 수로 이루어져 있는지 확인
- 1~9까지의 수 인가 &rarr; `Randoms.pickNumberInRange(1, 9)`을 사용하므로 구현 생략

### 2. 플레이어의 숫자 입력  
- `camp.nextstep.edu.missionutils.Console`의 `readLine()`을 활용
        

### 3. 입력한 수에 대한 결과 출력
- 같은 수가 같은 자리에 있을 경우 &rarr; 스트라이크
- 같은 수가 다른 다리에 있을 경우 &rarr; 볼
- 같은 수가 전혀 없을 경우 &rarr; 낫싱
- 잘못 입력했을 경우 `IllegalArgumentException` 발생
  - 3자리의 수가 아닌 경우
  - 서로 다른 숫자가 아닌 경우
  - 1~9 사이의 숫자가 아닌 경우
  - 숫자가 아닌 문자를 입력한 경우


### 4. 게임 종료
- `3스트라이크`일 경우 `3개의 숫자를 모두 맞히셨습니다! 게임 종료` 문구 출력
- `게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.` 안내 후, 사용자의 입력에 따라 게임 종료 여부 결정
  - 1 &rarr; 게임 재시작
  - 2 &rarr; 게임 종료
  - 잘못 입력한 경우 &rarr; `IllegalArgumentException` 발생
