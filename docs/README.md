# 미션 - 숫자 야구

## 🚀 구현할 기능 목록

1. 게임 시작을 알리는 문구 출력 기능
2. 게임 시작 후 플레이어의 입력 받기 전 컴퓨터가 생각하고 있는 숫자 설정 기능
3. 플레이어 숫자 입력 기능
4. 플레이어 숫자와 컴퓨터가 생각하고 있는 숫자의 비교 기능
5. 비교 결과 출력 기능
6. 게임 종료 후 게임 재시작 여부 (1/2) 결정 기능

## 🚨 예외 처리 사항

1. 플레이어 숫자의 중복
2. 플레이어의 입력이 숫자가 아님
3. 게임 종료 여부 시 입력 값이 1 또는 2가 아님
4. 플레이어의 숫자의 개수가 3개가 아님
