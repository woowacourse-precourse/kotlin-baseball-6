# 기능 목록

## 입력
- [ ] 게임 시작하면, 서로 다른 3자리 수 입력받기
- [ ] 게임이 끝난 후 재시작/종료 입력 받기

#### 예외
잘못된 값을 입력하면 `IllegalArgumetException` 발생 후 강제 종료
- [ ] 숫자가 아닐 때
- [ ] 같은 숫자를 입력했을 때
- [ ] 3자리의 수가 아닐 때
- [ ] 게임 끝난 후 입력 받은 값이 1 또는 2가 아닐 때

## 출력
- [ ] 3개 숫자 모두 맞혔을 때의 문구
- [ ] 볼이거나 스트라이크일 때의 문구
- [ ] 하나도 없을 때의 문구
- [ ] 게임 시작 문구
- [ ] 게임 재시작/종료 선택 문구

## 연산
- [ ] 랜덤한 수(1~9 중 서로 다른 3개의 수) 생성
- [ ] 스트라이크 및 볼 개수 카운트
