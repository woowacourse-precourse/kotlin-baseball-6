### 🚀 기능 요구 사항

- 랜덤한 숫자를 입력받아 야구게임을 실행 해야한다.
- 야구 게임은 서로 다른 3 자리 수를 맞추는 게임이다.
- 숫자와 자리 수가 맞으면 스트라이크, 숫자만 맞으면 볼, 전부 다르면 낫싱을 출력한다.
- 숫자를 맞히면 게임이 종료 된다.

## 🚀 기능 목록
