# 구현할 기능 목록

1. 컴퓨터가 세자리 수의 랜덤 값을 생성한다.
   - 1부터 9 사이의 수여야한다.
   - 중복이 없어야한다.
2. 사용자로부터 입력 값을 받는다.
   [잘못된 입력 값]
   - 3자리 미만인 경우
   - 숫자가 아닌 값의 경우
   - 자리수가 중복되는 경우
   - 1부터 9 사이의 수가 아닌 경우
3. 결과 값을 출력한다.
    - 입력값과 컴퓨터의 값을 비교한 후에 출력한다.
      - 같은 수가 같은 자리에 있으면 스트라이크, 같은 수가 있지만 다른 자리에 있으면 볼, 같은 수가 없으면 낫싱 출력
      - 숫자를 모두 맞히면 결과 출력 후 게임이 종료
4. 게임이 끝나면 새 게임을 시작을 하거나 종료한다.
