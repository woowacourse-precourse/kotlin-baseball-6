# 숫자 야구 게임 프로그램 구현

## 기능 목록
- [ ] 컴퓨터가 1~9 사이의 서로 다른 임의의 수 3개를 선택하는 기능
- [ ] 플레이어가 1~9 사이의 서로 다른 수 3개를 입력하는 기능
- [ ] 컴퓨터와 플레이어의 숫자를 비교해 결과(스트라이크, 볼, 낫싱)를 출력하는 기능
- [ ] 게임이 종료되면 게임을 다시 시작하거나 프로그램을 완전히 종료할 수 있게 선택하는 기능

## 예외 기능 목록
- [ ] 플레이어가 1~9 사이의 수 3개를 입력하지 않았을 때
- [ ] 플레이어가 중복된 수를 입력했을 때
- [ ] 1, 2를 제외한 값을 입력했을 때
