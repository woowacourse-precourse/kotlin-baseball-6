# 💻 구현 기능 목록
# 1주차 - 숫자 야구

1. 난수 생성 (answer)
   - 1-9로 이루어진 3자리 임의의 수 생성 
   - 중복 수 여부 확인

2. 게임 시작
   - 게임 시작 멘트 출력

3. 사용자 입력 (input)
    - 사용자 입력값 저장
    - 예외처리(IllegalArgumentException)
      - 입력 3자리 중 중복 발생 시
      - 입력 3자리 개수 미충족 시

4. 입력 검사 
    - input과 answer 스트라이크 개수 카운팅
    - strike == 3
      - 게임 종료
    - strike < 3
      - 전체 자릿수 별 검사 시작(볼 개수 검사)
5.  결과 출력
    - strike, ball, 낫싱 
6. 게임 종료 및 재시작 여부
    - 1 입력 시
      - 재시작
    - 2 입력 시
      - 시스템 종료
    - 예외처리(IllegalArgumentException)
      - 1, 2 외 입력 시
